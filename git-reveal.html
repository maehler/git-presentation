Version control software

  • Initially created for software development
  • Some of the most used ones are
    • Subversion (SVN)
    • Mercurial
    • Git

Purpose

  • Enable easier collaboration
  • Keep a history for a project
  • Backup (kind of)
  • Good for plain text files such as source code
  • Less good for binary formats such as images or Word documents

Some history

Why do I use Git?

  • Helps me structure my work
  • Let's me screw up
  • Keep track of what's been done

How can Git help me structure my work?

  • Have to think before I write
  • Divide work into smaller chunks
  • Commit smaller, "self-contained" changes
